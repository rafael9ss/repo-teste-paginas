**URL do Website:** Será fornecida após a publicação

## Powered by Manus

Esta landing page foi desenvolvida com tecnologias de ponta para garantir performance e conversão máximas:

**Frontend:** React 19 + TypeScript + Tailwind CSS 4 + shadcn/ui  
**Design System:** Sistema de cores customizado (preto, laranja, branco) com componentes otimizados para conversão  
**Deployment:** Infraestrutura auto-escalável com CDN global para carregamento instantâneo em qualquer lugar do mundo  

A arquitetura moderna garante tempos de carregamento ultra-rápidos, design responsivo perfeito em todos os dispositivos e uma experiência de usuário fluida que maximiza as taxas de conversão.

## Usando Seu Website

A landing page foi estruturada para guiar o visitante através de uma jornada de conversão otimizada:

**Navegação e Conversão:** Ao acessar a página, o visitante encontra imediatamente a proposta de valor no Hero Section. Clique em qualquer botão "QUERO PARAR DE PERDER DINHEIRO AGORA" → A página rola suavemente até a seção de planos → Escolha entre "Plano Básico (R$ 10)" ou "Plano Completo (R$ 29,90)" → Clique no botão correspondente para iniciar a compra.

**Prova Social:** Role para baixo após o Hero → Veja estatísticas de sucesso (+1.500 alunos, +R$ 10M em vendas) → Explore os depoimentos reais em imagens de WhatsApp que comprovam os resultados.

**Comparação de Planos:** Na seção "ESCOLHA SEU ATALHO PARA O SUCESSO" → Compare lado a lado os recursos do Plano Básico vs Completo → Note o selo "MAIS ESCOLHIDO - 90% ESCOLHEM ESTE" destacando o Plano Completo → Veja claramente o que está incluído em cada plano com ícones de check (✓) e X.

## Gerenciando Seu Website

**Atualização de Conteúdo:** Para modificar textos, preços ou adicionar novos depoimentos, acesse o painel "Code" no Management UI → Navegue até `client/src/pages/Home.tsx` → Edite o conteúdo diretamente ou peça ajuda ao Manus AI.

**Substituição de Imagens:** Para trocar o logo ou depoimentos, acesse "Code" → Navegue até `client/public/` → Faça upload das novas imagens com os mesmos nomes ou atualize as referências no código.

**Publicação:** Após fazer alterações, clique no botão "Publish" no canto superior direito do Management UI para tornar as mudanças visíveis ao público.

**Analytics:** Acesse o painel "Dashboard" para visualizar métricas de visitantes (UV/PV) e acompanhar o desempenho da sua landing page.
